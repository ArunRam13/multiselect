Multiselect

This is a small jQuery plugin that helps you improve the user experience regarding the use of multiple cross selects.

It is very easy to install and can be easily customized because it has callbaks for many events, such as :

  • beforeMoveOneToRight
  • beforeMoveOneToLeft
  • beforeMoveAllToRight
  • beforeMoveAllToLeft
  • afterMoveOneToRight
  • afterMoveOneToLeft
  • afterMoveAllToRight
  • afterMoveAllToLeft

It also has a sort function that can be defined in order of your needs for each multiple select.

You can fully customize it via CSS, you can extend it and have as many multi selects as you want on the page.

<script type="text/javascript">
$(document).ready(function(){
	// initialize the multiselect for #multiselect element
	$('#multiselect').multiselect();
});
</script>
var options = {
	startUp: function( $left, $right ){			
		$right.find('option').each(function(index, option){
			$left.find('option[value="' + option.value + '"]').remove();
		});
	},
	sort: false,
	right: '#to2',
	rightAll: '#rightAll2',
	rightSelected: '#rightSelected2',
	leftAll: '#leftAll2',
	leftSelected: '#leftSelected2'
};

$('#from').multiselect(options);
